# Projeto para atividade de Clean Code - Gerenciador de Games
 
## 🔐 Pré requisitos

<a href="https://vitejs.dev/">Vite</a> &nbsp;
<a href="https://nodejs.org/en/">Node.js</a> &nbsp;

## Instalação do projeto e dependências

npm i

## Executando a aplicação

```bash
# development
$ npm i
$ npm run dev
$ o + enter
```
